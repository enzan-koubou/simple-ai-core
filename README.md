
Simple AI Core
==============


Automatically connect to the database and execute SQL query.  


Dependencies
------------

GoLang https://go.dev/doc/  
python 3 https://docs.python.org/3/ (test with 3.11.2)  


Execute
-------

`go run server.go`  

Query  
GET http://localhost:8080/query?q=some_query_text
